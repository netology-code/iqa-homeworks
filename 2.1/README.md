* Сценарии нужно создать по шаблону: https://docs.google.com/spreadsheets/d/1sUZ1GbLdJLubR1Px_4YCy5BGdhFT-FmxHiR-luC9_X8/edit?usp=sharing (создайте копию файла и заполните его).
